Note:
=====
Before run the setup program to install newer driver, please uninstall the older driver
and setup program that user had installed before. User can check the list in "Control Panel" -> "Programs" ->
"Programs and Features". If there is a AX88178 driver in the list, uninstall it
before run the setup program.
